# 🕹️ Jokenpô Moderno (Pedra, Papel e Tesoura)

Um jogo interativo de Pedra, Papel e Tesoura desenvolvido com foco em **UX (Experiência do Usuário)**, rapidez de resposta e design responsivo. O projeto foi construído para ser leve, intuitivo e funcional em qualquer dispositivo.

https://guilhermejobs.github.io/Jokenp-/

## 🚀 Tecnologias
O projeto utiliza a "trindade" essencial da web, mantendo o código modular e fácil de manter:
* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização moderna com Grid/Flexbox e design responsivo.
* **JavaScript:** Lógica de processamento e manipulação dinâmica do DOM.

## ✨ Funcionalidades
* Interface Baseada em Cards: Seleção visual intuitiva com feedback de seleção.
* Imagens Dinâmicas: Sistema preparado para carregar assets diretamente via GitHub.
* Design Responsivo: Adaptado para telas de smartphones, tablets e desktops.
* Fluxo de Jogo Fluido: Telas de seleção e resultado separadas para melhor foco do usuário.

## 🛠️ Como rodar o projeto
1. Clone o repositório:
git clone [https://github.com/GuilhermeJobs/Jokenp-git](https://github.com/GuilhermeJobs/Jokenp-.git)
2. Abra o arquivo index.html em seu navegador de preferência.
